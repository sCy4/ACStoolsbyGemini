清關報告幫手

自動化清關查單與回填工具 — 使用說明書

版本　v1.1.0 作者　蘇宗彥（ACS 台北站） 適用對象　ACS 清關作業人員

01這是做什麼的？

一個跑在背景的 Windows 小工具，幫你自動處理清關報告中重複又繁瑣的查單動作。 在報告中選取單號 → 長按右鍵 → 選功能，工具會自動切到物流系統查詢、抓取狀態，再透過雲端寫回清關報告。

它能解決什麼？

以前現在
一筆一筆複製單號、貼到物流系統、查 EZWAY 狀態、回到報告手動填上「資料相符」「Y/N」與分配人員⋯⋯ 選取整欄單號 → 長按右鍵選功能 → 倒杯水回來，看統計報告。
核心理念　不打擾你平常使用 Chrome：短按右鍵 = 正常右鍵；長按右鍵 = 工具選單。

02四種功能（使用方式及情境）

🔍
查詢單筆
情境：客服臨時被問「這筆單現在狀態？」、想快速確認一張單的明細。
  1. 在任何地方選取一個單號文字（可以是清關報告、聊天訊息、Excel⋯⋯）
  2. 在 Chrome 中長按右鍵，選擇「查詢單筆」
  3. 工具自動切到「物流管理系統」並把該單號查出來，完成後游標恢復正常
🔄
更新申報狀態
情境：每日例行清關報告維護 — 批次查多筆單號的 EZWAY 狀態並自動回填。
  1. 在「清關報告」中選取一整欄的單號（可選 5 筆、50 筆、500 筆）
  2. 長按右鍵，選擇「更新申報狀態」
  3. 畫面上方顯示進度（例：▶️ 腳本運作中 [進度 3 / 50]）
  4. 跑完後自動切到「清關報告」分頁，透過雲端寫回「資料相符」「有效」
  5. 完成後彈出統計報告（總執行 / 已相符 / 已上傳 / 其他 / 狀態更改數）
注意　寫回階段會顯示「⏳ 修改表單資料中」，此時可以做別的事，但不要關掉 Chrome。
👥
標記 Y/N 與分配人員
情境：需要把當天未相符的單號分派給組員追蹤處理。
  1. 點選功能後跳出輸入視窗，自動帶入上次或預設名單（萍, 富, 蓁, 姿, 彥, 潔）
  2. 修改名單後按 OK（會自動記住，存到 腳本設定檔-清關名單預設分配人員.txt）
  3. 只想標 Y/N 不分配人員 → 把名字全部清空再按 OK
  4. 後續流程與「更新申報狀態」相同，雲端會額外寫入 Y/N 與分配人員欄
分配邏輯　名單為「萍, 富, 蓁」，這次有 7 筆需要分配 → 萍、富、蓁、萍、富、蓁、萍（輪流；已相符的單不分配）。
⚡
標記重複資料
情境：報告中可能有重複單號，需要一鍵把它們標色顯示。
  1. 切到「清關報告」分頁
  2. 長按右鍵，選擇「標記重複資料」
  3. 工具會送出 Ctrl + Alt + Shift + 1，觸發試算表內預先綁定的巨集
前提條件　試算表必須先綁定對應巨集到 Ctrl + Alt + Shift + 1。此功能只能在 Chrome 視窗中使用。

運作中可以用的按鍵

按鍵動作
長按右鍵（> 0.3 秒）開啟工具選單（短按 = 正常右鍵，不會誤觸）
Esc暫停 / 恢復 — 做完事回到原畫面再按 Esc 繼續
F8結束工具（畫面出現「再見」後 2 秒關閉）

03如何安裝？

環境需求

  • Windows 10 或 11
  • Google Chrome（已登入物流系統）
  • 網路連線（雲端寫入與自動更新用）
  • 必須開好兩個分頁：
    ① 物流管理系統
    ② 清關報告

安裝步驟

  1. 取得 清關報告幫手.exe，放在固定資料夾，例如 D:\工具\
  2. 雙擊執行檔，畫面右上出現「✅ 腳本已啟動」即代表運作中
  3. 確認 Chrome 已開好兩個分頁並完成登入
  4. 在 Chrome 中長按右鍵 → 出現工具選單即安裝成功
提醒　分頁標題必須包含「物流管理系統」與「清關報告」這兩個關鍵字，工具才找得到。 工具不會幫你登入，請事先登好。自動更新已內建：啟動後 5 秒會背景檢查 GitHub，有新版下次重啟自動套用，全程無感。

04錯誤訊息對照表

看到錯誤對話框時，對照此表找原因與解法：

錯誤訊息 原因 解決方式
沒有偵測到你要執行的單號沒有先選取單號就執行先反白選取單號文字再按右鍵
找不到「物流管理系統」的網頁Chrome 沒開或沒登入打開分頁並登入
「物流管理系統」網頁異常物流系統頁面結構異動重新整理頁面，再不行請聯絡作者
清關報告內沒有找到你所執行的單號你選的單號不在清關報告中確認單號是否真的在報告內
清關報告的雲端指令碼沒有回應雲端伺服器異常或網路斷線等 30 秒重試；持續失敗請聯絡作者
這個功能只可以在清關報告中使用在非 Chrome 視窗按了「標記重複資料」切回 Chrome 再執行
單號好像不對（需要 > 3 字元）選取內容全是太短的單號重新選取正確單號
這超過了腳本能力範圍發生未預期錯誤重試一次；持續失敗請附畫面截圖聯絡作者

05常見問題

右鍵按了沒反應？
你可能是「快點」了，請改成按住右鍵 0.3 秒以上。這個設計是為了不干擾你正常使用 Chrome。
可以同時開好幾個 Chrome 視窗嗎？
可以，工具會自動掃描所有 Chrome 視窗，找到含有對應分頁的那一個。
執行到一半可以做別的事嗎？
沙漏游標時不行（滑鼠被鎖）；正常游標時可以；想中途切換請按 Esc 暫停。
剪貼簿原本的內容會不見嗎？
不會，工具會自動備份你的剪貼簿，執行完還原回去。
分配人員設定要去哪裡改？
① 下次執行功能時在輸入框內直接修改（自動記憶）。② 直接編輯 腳本設定檔-清關名單預設分配人員.txt。
100 筆要跑多久？
每筆約 3–8 秒，100 筆約 5–10 分鐘。中途可隨時 Esc 暫停。
跑到一半單筆查詢失敗？
工具會自動把該筆標為「失敗」，繼續處理下一筆，不會中斷整批流程。
要關閉工具怎麼做？
按 F8，畫面出現「再見」後 2 秒自動關閉。
